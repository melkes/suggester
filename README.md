## Setup/Installation Requirements

* Clone this repo and open index.html

## Known bugs
* I was not able to fully figure out how to collect process inputs from multiple groups of radio button form inputs. Therefore, the last answer of the last question is in a group with the answers to the first question. Unfortunately my attempts at including multiple input names made the whole thing break.

## Needed improvements:
* This project only has three functioning answers: either answer on the first question, and the second answer on the last question. I would have liked to actually have each question contribute to the result.
* The design didn't get a whole lot of attention and is painfully dull.

2. Do you want to make websites? 
    - Yeah but nothing too crazy __(HTML/CSS)__
    - Modern, interactive ones __(HTML and Javascript)__ 

3. Do you want to work in “business”?
    - __Python or Java__

4. Something something databses 
    - __(SQL)__
